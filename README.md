# Lista de Tarefas ✅

Aplicação simples de lista de tarefas feita com React e testada com React Testing Library + Jest.

## Requisitos

- Node.js (versão 18+ recomendada)
- npm ou yarn

## Instalação

Clone o repositório e instale as dependências:

```bash
npm install
# ou
yarn
```

## Executar o projeto

O projeto geralmente estará disponível em:
http://localhost:5173

```bash
npm run dev
# ou
yarn dev
```

## Executar os testes

```bash
npm test
# ou
yarn test
```

## O que foi testado

- Componente NovaTarefa (input, botão e submissão)
- Hook useContadorDeTarefas de forma isolada
- Renderização da página com lista de tarefas
- Função de adicionar e remover tarefas

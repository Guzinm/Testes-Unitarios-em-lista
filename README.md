# Lista de Tarefas ✅

Aplicação simples de lista de tarefas feita com React e testada com React Testing Library + Jest.

## O que foi testado

- Componente NovaTarefa (input, botão e submissão)
- Hook useContadorDeTarefas de forma isolada
- Renderização da página com lista de tarefas
- Função de adicionar e remover tarefas

## Feita uma pipeline completa de CI/CD testando:

- Instalando dependencias
- Rodando lint
- Rodando testes
- Rodando build
- Rodando deploy na vercel
